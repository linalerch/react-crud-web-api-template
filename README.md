### in node-express-mongodb

#### first time
```
npm install
```

#### Run
```
node server.js
```

### in react-crud


#### first time
```
npm install
```

#### Compiling + run

```
npm start
```

Open [http://localhost:8081](http://localhost:8081) to view it in the browser.

The page will reload if you make edits.
